git 브랜치 자동 머지의 장단점은 무엇일까?
- 장점 : 빠르게 머지를 진행할 수 있다.
- 단점 : 원치 않게 소스코드가 합쳐질 수 있다.

remote와 origin 저장소의 차이점은 무엇인가?
- remote는 원격이라는 의미로 github 등의 클라우드상에 존재하는 저장소 전반을 의미하고, origin은 내 로컬 repository와 연결할 특정의 원격 저장소를 이름을 붙여 식별한 것이다.
